# 📝 To-Do App

A simple and clean To-Do List application built with React. This project was created as part of my journey learning React from scratch — covering components, state management, hooks, and local storage persistence.

## ✨ Features

- ➕ Add new tasks
- ❌ Delete tasks
- 💾 Tasks persist using LocalStorage (even after page refresh)
- 🎨 Clean and simple UI
- ⚡ Built with React + Vite for fast performance

## 🛠️ Tech Stack

- **React** – Frontend library for building UI
- **Vite** – Fast build tool and dev server
- **CSS** – Custom styling
- **LocalStorage API** – For persisting tasks in the browser


## 🚀 Getting Started

### Prerequisites
- Node.js installed on your machine

### Installation

1. Clone the repository
```bash
git clone https://github.com/Jaydip-Nagdakiya/todo-app.git
```

2. Navigate to the project folder
```bash
cd todo-app
```

3. Install dependencies
```bash
npm install
```

4. Run the development server
```bash
npm run dev
```

5. Open `http://localhost:5173` in your browser
